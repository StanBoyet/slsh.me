Notes

What the
chosen direction covers.

  • Links index — fully populated app screen: 10 link rows (campaigns, custom domain, password, expired, OG, limit, API, archived, live-flash), header stats with gradient corner orbs, dev surface stickers, filters, active-campaign banner, empty state.
  • Landing page — full marketing page: hero (Paste · Slash · Share, Migra), trust bar, Analytics + Custom Domains, 6-card feature grid (Beam orbs + Splash flat icons), three personas with marker-highlighted headlines, FAQ, changelog, light final CTA.
  • Typography rule. Migra (display serif) renders only the hero h1. Every other H2 / H3 / heading uses General Sans. Mono for slugs and code chips.
  • Beam injections. Gradient corner orbs on KPI tiles + feature cards, gradient stroke on the analytics chart, illustrated OG preview, light (not dark) final CTA. Everything else stays Splash — ink borders, marker highlights, sticker pills, hard-shadow card pops, tilted cards that straighten on hover.
  • No app code touched. Static HTML in public/brand/. The live landing page at / is unchanged.
