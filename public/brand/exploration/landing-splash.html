Questions

Is slsh.me really free?
Yes, completely. No paid tiers. Custom domains, analytics, all link features — included.
How is this different from Bitly?
Bitly paywalls custom domains, expiry, and password protection. Here, all of that is free.
What analytics do I get?
Clicks, uniques, country, city, browser, OS, device, and referrer. All real-time, all first-party. No cookies, no external scripts.
Is there an API?
REST API with token auth. Create, update, delete, list links. Grab your token from settings.
Will my links break if slsh.me goes down?
Multi-region deployment, so downtime is unlikely. And it’s open source — you can self-host anytime.
